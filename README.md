# 📋 Syntecxhub To-Do App

A modern Android To-Do List application built using Kotlin and XML as part of the Syntecxhub Android Development Internship.

---

## ✨ Features

- ➕ Add Tasks
- ❌ Delete Tasks
- ✅ Mark Tasks as Completed
- 📦 Local Data Storage using SharedPreferences
- 📜 RecyclerView Task Listing
- 🌙 Modern Dark UI
- 📊 Task Counter
- 📭 Empty State Message

---

## 🛠 Tech Stack

- Kotlin
- XML Layouts
- Android Studio
- RecyclerView
- SharedPreferences

## 🚀 How to Run

1. Clone this repository
2. Open project in Android Studio
3. Sync Gradle files
4. Run the app on Emulator or Android Device
